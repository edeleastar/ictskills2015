ICT Skills 2015

This is a 'taster' for the the ICT Skills 2015 programme at Waterford Institute of Technology. It consists of 6 topics, each topic representing 1 weeks tuition material for a single week. The first 3 topics are from the Web Development module, the second three are form the Programming module. They are designed to run concurrently.
