Prog 1: Introduction

Introduction

Before we start learning to program, we need to discuss the tools we are using, the role of these tools in our work and the nature of programs generally.

Objects

We are learning Java - an Object Oriented Language. Here we explore the nature of an object and perform some simple experiments on our tools to gain an initial understanding.

Lab: Lab-01-A

  • Acquire experience in the use of the BlueJ Integrated Development Environment (IDE) application which will be used as the primary tool in learning the Java programming language throughout much of this course.

  • An introduction to object oriented programming (OOP).

  • Basic explanation of key concepts of OOP such as classes, objects, methods and data types.

  • Create graphical representations of simple shape objects such as triangles, circles and rectangles.

Lab: Lab-01-B

  • Continuation of previous lab

  • Examine Java code that generates objects that we have created earlier

  • Acquire a basic understanding of

    • methods
    • parameters
    • state

  • Gain experience in using the BlueJ terminal window

  • By the end of the lab you will be comfortable solving some simple exercises such as writing Java code to calculate properties of some of the shape objects, for example, the area and perimeter of a circle, the volume of a cone.
