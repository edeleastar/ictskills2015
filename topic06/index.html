Prog 3: Objects

Abstraction

All programmes are built using abstractions - which can be considered simplified perspectives on the problem we are trying to solve.

Programs

Computer programmes realise useful abstractions when solving specific problems. Here we explre a simple clock program, and examine it from a number of angles.

Lab: Lab-03-A

  • Use set of shapes objects to manually create a house object
    • Use existing Square, Circle and Triangle classes
    • Manually assemble component set to create initial design
    • Display initial design
  • Write a new class RectangleBJ by refactoring Square class
  • Refactor Circle and Triangle classes

Lab: Lab-03-B

  • Demonstrate abstraction and modularization
    • Develop a new class Tree using RectangleBJ and Triangle classes
    • Develop a new class Picture containing House, Circle (sun) and Tree fields

Lab: Lab-03-B

Based on BlueJ ClockDisplay (Objects First with Java Edition 3)

  • Progressively develop
    • NumberDisplay: a class to display a pair of digits to represent hours and minutes
    • ClockDisplay: a class to the operation and display of a clock
  • Use the existing BlueJ class Clock to drive the ClockDisplay objects.
  • Perform some basic testing.
